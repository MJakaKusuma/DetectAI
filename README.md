# DetectAI — Hybrid AI Text Classifier

DetectAI is an end-to-end, production-ready machine learning web application designed to distinguish between human-written texts (sourced from Wikipedia) and AI-generated texts (generated by Gemma 4) in Bahasa Indonesia. 

The system leverages a **hybrid feature extraction** approach, combining lexical content representation (**TF-IDF**) with structural writing style metrics (**Stylometry**), classified using a well-regularized **Logistic Regression** model.

---

## 🚀 Key Features

### 👤 User Dashboard (Interactive Diagnostics Lab)
*   **Real-time Text Analyzer:** Inspects Indonesian texts with dynamic word and character counters.
*   **Stylometric Benchmarking:** Compares the input text's sentence length, lexical diversity, and punctuation density against standard AI and Human baselines in real-time.
*   **Lexical Scan (TF-IDF Clues):** Automatically highlights and identifies vocabulary keywords statistically favored by generative language models.
*   **Printable PDF Reports:** Generates professional, print-ready diagnostic reports using native browser print styles (zero-dependency).
*   **Human-in-the-Loop Feedback:** Allows users to submit classification corrections directly to the database for model auditing.

### 🛠️ Administrator Panel (MLOps Control Center)
*   **Analytics Dashboard:** Displays system statistics, including total predictions, total users, active model accuracy, and total user feedback.
*   **Interactive Dataset Merger:** A client-side preprocessing tool that merges separate `human.csv` and `ai.csv` files, enforces strict class-balance validation rules (rejects unbalanced datasets to prevent bias), and uploads the merged dataset to the server.
*   **Continuous Retraining:** Triggers an automated training pipeline on the server that rebuilds the TF-IDF vectorizer and Logistic Regression model, evaluates performance, and updates the active model registry.
*   **Model Registry & Rollback:** Displays historical model versions with their accuracy and F1-scores, allowing admins to activate or rollback to older model versions with a single click.

### 🔌 Developer Center (API Documentation)
*   Provides complete integration documentation for third-party systems.
*   Includes ready-to-use request/response JSON schemas and copy-pasteable code snippets in **cURL**, **Python**, and **JavaScript**.

## 📊 Methodology & Feature Engineering

DetectAI transforms raw text into a **1,003-dimensional numerical vector** before feeding it into the Logistic Regression classifier:

1.  **Lexical Features (1,000 Dimensions):** Extracted using a `TfidfVectorizer` limited to the top 1,000 most significant terms across the corpus.
2.  **Stylometric Features (3 Dimensions):**
    *   *Average Sentence Length:* Total words divided by total sentences (measures structural monotony).
    *   *Lexical Diversity (Type-Token Ratio):* Total unique words divided by total words (measures vocabulary richness).
    *   *Punctuation Density:* Total punctuation marks divided by total words (measures grammatical strictness).

Both feature sets are concatenated horizontally (`np.hstack`) to create a balanced hybrid input representation.
